Computer and Internet Browser Settings

The New Employee Orientation requires the following computer and Internet browser settings:

Computer screen resolution:  1024x768
Internet browser type:  Internet Explorer (IE), version 6 or higher
IE pop-up setting: Must be set to allow pop-ups

Setting Screen Resolution

To properly view the New Employee Orientation, your computer's screen resolution must be set at 1024x768. To set your screen resolution to 1024x768, do the following:
  1. Select the Microsoft Windows Start button and select "Settings," then "Control Panel" from the menus. The "Control Panel" window will appear.
  2. Double-click on "Display." The "Display Properties" window will appear. 
  3. Select the "Settings" tab.
  4. In the "Screen Resolution" area, check to see if your resolution is set to 1024x768.  If it's not, take note of your current resolution so you can return to it, then drag the setting indicator to the right or left (depending on what your current resolution is) until you reach 1024 by 768 pixels. 
  5. Select the Apply button, then select the OK button.

(Follow the same steps to return to your preferred screen resolution.)

Allowing Pop-Ups

The New Employee Orientation contains hyperlinks to useful Web sites. In order for many of these hyperlinks to function properly, Internet Explorer must be set to allow pop-up windows; they cannot be blocked. To set Internet Explorer to allow pop-up windows, do the following:
  1. In the Internet Explorer window, select "Tools" from the menu bar at the top of the window.  The "Tools" drop-down menu will appear.
  2. Select "Internet Options." The "Internet Options" window will appear.
  3. Select the "Privacy" tab.  In the "Pop-up Blocker" area, make sure the "Block pop-ups" check box is not checked.
  4. Select the OK button.

NOTE: All third-party pop-up blockers (Yahoo, Google, etc.) must also be set to allow pop-ups.

About the House Topic

The About the House topic is divided into two sections:

  • History of the House—This section provides a brief history of the U.S. House of Representatives
  • Today's House—This section provides an overview of the current House of Representatives

Navigation

There are two methods for navigating through the About the House topic (both methods are indicated below in Figure 3):

  • Side menu links—The side menu links are located on the left side of the screen and allow you to jump to specific subjects within the About the House sections
  • Right and left arrows—The right and left arrows are located on the lower right corner of the screen and allow you to move page-by-page through the About the House sections

On-screen prompts

At the bottom of each screen in the About the House topic is an on-screen prompt (indicated below in Figure 3).  This prompt provides information on how to progress through the topic.  If you find yourself unsure of what to do next, simply look to the on-screen prompt for assistance.

    OR
  • Select the "Main Menu" button, located in the lower left corner of the screen (indicated below in Figure 3)

Welcome to the House Topic

The Welcome to the House topic provides information regarding all aspects of being a House employee—from how to get your security clearance to where to get your shoes shined.  The topic is presented via an interactive rotating "dome."

"Dome" Sections

The Welcome to the House "dome" consists of six sections:

  • Benefits
  • Campus Services
  • Health and Safety
  • Professional Development
  • Security
  • Employee Rights & Responsibilities

Each section contains multiple subjects, which, when selected, provide useful information for new House employees.

"Dome" Columns

In addition to the six section, the Welcome to the House "dome" also contains five columns that can be selected for more information:

  • Members' Websites
  • HouseSmart
  • HouseNet
  • Did You Know?
  • Acronym List

Navigation

The sections of the dome can only be accessed if they are facing you (i.e., on the front side of  the circle).

To rotate the dome to access a section, simply select the right or left arrow located on the navigation bar below the dome (indicated below in Figure 4):

  • Select the right arrow to rotate the dome in a counter-clockwise direction
  • Select the left arrow to rotate the dome in a clockwise direction

Active sections (i.e., sections that can be selected) are indicated by a highlight when the mouse is rolled over them.

Accessing Subjects Within a Section

To access a subject within a section, use the left or right arrow to rotate the dome until the section containing the desired subject is facing you, then simply select the desired subject. 

Returning to the Main Menu

To return to the Main Menu, do one of the following:

  • Close the Welcome to the House browser window
    OR
  • Select the "Main Menu" button, located on the left of the navigation bar (indicated  below in Figure 4)

Figure 4 – Welcome to the House screen

Text-Only Version

Located in the lower left corner of the Welcome to the House window is a link to a text-only version of the interactive dome (indicated above in Figure 4).  Selecting this link opens a new browser window containing the text-only version of the topic.

The text-only version of the Welcome to the House topic presents the exact same information as the interactive dome version, but sections and subjects are accessed by selecting links on a side menu (indicated below in Figure 5), as opposed to selecting subjects on the dome.

Hyperlinks

There are two types of hyperlinks used in the New Employee Orientation:

  • Hyperlinks to pop-up windows which contain additional information
  • Hyperlinks to Web sites outside of the New Employee Orientation

Hyperlinks to Pop-Up Windows

Hyperlinks to pop-up windows will open a pop-up window within the New Employee Orientation.  To close the pop-up window, simply select the Close button located within the pop-up.

Hyperlinks to Web Sites

Hyperlinks to Web sites will open a new browser window displaying the Web site in front of the New Employee Orientation window. 

To return to the New Employee Orientation window, do one of the following:

  • Close the browser window displaying the Web site
    OR
  • Minimize the browser window displaying the Web site
